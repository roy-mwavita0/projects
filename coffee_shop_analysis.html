Install & Load Libraries

Install pacman in the console (install.packages(pacman). Pacman is library used to efficiently manage libraries. It loads the libraries and installs them first if they are not.

pacman::p_load(
  tidyverse, tidylog, janitor, readxl, magrittr, skimr
)

Import Data

Use read_excel to load the dataset by copy pasting the path.

df <- read_excel("D:/Personal/Projects/coffee_shop/coffee_shop_survey.xlsx")

Clean Column Names

Janitor simplifies the column names by removing any symbols and spaces, then converting them to lower case. This is optional. You can choose not to use the funtion.

df_clean <- df %>% 
  clean_names()

3. Do loyalty members spend more time at the coffee shop than non-members?

df_clean %>% 
  group_by(loyalty_member) %>% 
  summarise(avg_time_spent = mean(time_spent_min)) %>% 
  ggplot(aes(x=loyalty_member, y= avg_time_spent, fill = loyalty_member)) + 
  geom_col() +

# Perform t-test

##          30.53333          35.67273

T-test results suggest that loyalty members spend slightly more time at the coffee shop than non-members (mean: 35.67 min vs.30.53 min). However, the p-value (0.07917) is above the common significance threshold of 0.05, meaning While this suggests that loyalty members might spend a bit more time at the coffee shop, its not strong enough evidence to confirm a real pattern.

4. Are customers who would recommend the coffee shop more likely to be loyal members?

# Create contingency table
chi_square_data <- table(df_clean$would_recommend, df_clean$loyalty_member)

##       No Yes

## 
##  Pearson's Chi-squared test with Yates' continuity correction
## 
## data:  chi_square_data
## X-squared = 4.4325, df = 1, p-value = 0.03526

Chi-Square test results suggest that customers who would recommend the coffee shop are more likely to be loyalty members. Since the p-value (0.03526) is below 0.05, we reject the null hypothesismeaning theres a statistically significant relationship between recommendation and loyalty membership.
